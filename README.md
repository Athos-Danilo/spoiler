# Spoiler - Quiz de Filmes & Séries

![Badge em Desenvolvimento](http://img.shields.io/static/v1?label=STATUS&message=EM%20DESENVOLVIMENTO&color=GREEN&style=for-the-badge)

O **Spoiler** é um Web App de Quiz estilo "Arcade" focado no universo de Filmes e Séries. O projeto foca na gamificação, pressão de tempo e gestão de recursos (pontos), testando tanto o conhecimento quanto a estratégia do usuário.

## Stack Tecnológica

### Frontend
* **HTML5 & CSS3:** Sem frameworks, focando em animações nativas e responsividade.
* **JavaScript (Vanilla):** Lógica pura para manipulação do DOM e consumo de API (Fetch).

### Backend (API REST)
* **Node.js & Express:** Servidor robusto e escalável.
* **MongoDB & Mongoose:** Banco de dados NoSQL para flexibilidade de schema.
* **JWT (JSON Web Token):** Autenticação segura via tokens.
* **BcryptJS:** Criptografia de senhas.
